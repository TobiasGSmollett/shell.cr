# shell

Small simplistic wrapper for executing shell commands in Crystal:

- run command, exit
- if it fails
- return STDOUT of the command.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  shell:
    github: [your-github-name]/shell
```


## Usage


```crystal
require "shell"
```
